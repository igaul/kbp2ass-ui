## kbp2ass converter ui

[live](https://kbp2ass.livekj.com/)

### build requirements:

- node >= 16

  ```
  curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
  ```

- rust
  ```
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  ```

### run

#### web

```
pnpm dev
```

#### app

```
cargo tauri dev
```
